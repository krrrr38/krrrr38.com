# krrrr38

## Setup

- initial setup

```sh
> vue create krrrr38 // TypeScript, vuex, router, TSLint, SASS
> firebase init // functions, hosting, firestore
```

## Development

- hosting

```sh
> nvm use v10.5.0
> yarn install
> yarn serve // open http://localhost:5000
```

- functions
```sh
> nvm use v6.11.5 // Cloud Functions support node version
> yarn serve
```

## Deploy

```sh
> npm run deploy
```
